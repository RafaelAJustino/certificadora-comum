Comandos de Inserção

Os comandos de inserção em SQL são utilizados para adicionar novos registros a uma tabela em um banco de dados. Eles permitem a inclusão de dados de forma estruturada e consistente, seguindo o esquema pré-definido da tabela. Os comandos de inserção mais comuns são o INSERT INTO e o INSERT SELECT.

Inserção

Exemplo do comando de INSERT:

                                                
INSERT INTO nome_da_tabela (coluna1, coluna2, coluna3, ...)
VALUES (valor1, valor2, valor3, ...);
INSERT possibilita a inserção de dados em uma tabela.
Exemplo do comando de INSERT SELECT:

                                                
INSERT INTO tabela_destino (coluna1, coluna2, ...)
SELECT coluna1, coluna2, ...
FROM tabela_origem
WHERE condição;
VALUES (valor1, valor2, valor3, ...);
INSERT SELECT possibilita a inserção de dados em determinada tabela a partir de uma consulta.
