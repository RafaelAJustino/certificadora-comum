Comandos de Exclusão

Os comandos de exclusão são usados para remover objetos de um banco de dados. O comando mais comum é o DROP TABLE, que remove uma tabela do banco de dados. O DROP DATABASE é usado para remover um banco de dados inteiro. Ao usar esses comandos, é importante ter cuidado para não excluir objetos acidentalmente, pois a exclusão é permanente e não pode ser desfeita.

Exclusão

Exemplo de comando DROP DATABASE: 

                                                
DROP DATABASE CompanyDB;
O comando DROP DATABASE, excluir o banco de dados.
Exemplo de comando DROP TABLE: 

                                                
DROP TABLE Employees;
O comando DROP TABLE, que remove uma tabela do banco de dados.
Exemplo de comando DELETE: 

                                                
DELETE FROM Employees WHERE EmployeeID = 1;
O comando DELETE, que remove dados de um tabela.
