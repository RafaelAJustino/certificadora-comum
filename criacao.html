Comandos de Criação

Os comandos de criação são usados para criar objetos em um banco de dados, como tabelas, bancos de dados, índices, visões etc. O comando mais comum para criar uma tabela é o CREATE TABLE, que define a estrutura da tabela, como nome, colunas, tipos de dados, restrições etc. O comando CREATE DATABASE é usado para criar um novo banco de dados. Ao criar objetos em um banco de dados, é importante definir as restrições e as regras de integridade dos dados.

Criação

Exemplo de comando CREATE DATABASE:

                                                
CREATE DATABASE CompanyDB;
O comando CREATE DATABASE é usado para criar um novo banco de dados. Ao criar objetos em um banco de dados, é importante definir as restrições e as regras de integridade dos dados.
Exemplo de comando CREATE TABLE:
                                                
CREATE TABLE Employees (   
   EmployeeID INT PRIMARY KEY,
   FirstName VARCHAR(50),
   LastName VARCHAR(50),
   Age INT,
   DepartmentID INT,
FOREIGN KEY (DepartmentID) REFERENCES Departments(DepartmentID)
           );
