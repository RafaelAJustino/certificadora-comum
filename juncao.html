Comandos de Junção

Os comandos de junção são usados para combinar dados de duas ou mais tabelas em um único conjunto de resultados. O INNER JOIN é usado para retornar apenas as linhas que têm uma correspondência nas duas tabelas. O LEFT JOIN e o RIGHT JOIN são usados para retornar todas as linhas da tabela à esquerda ou à direita, mesmo que não haja correspondência na outra tabela. O FULL OUTER JOIN é usado para retornar todas as linhas de ambas as tabelas, independentemente de haver correspondência.

Junção

Exemplo de comando INNER JOIN:

                                                
SELECT Orders.OrderID, Customers.CustomerName
FROM Orders 
INNER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;
O INNER JOIN é usado para retornar apenas as linhas que têm uma correspondência nas duas tabelas.
Exemplo de comando LEFT JOIN:
                                                
SELECT Customers.CustomerName, Orders.OrderID   
FROM Customers
LEFT JOIN Orders ON Customers.CustomerID = Orders.CustomerID;
O LEFT JOIN é usados para retornar todas as linhas da tabela à esquerda.
Exemplo de comando RIGHT JOIN:
                                                
SELECT Customers.CustomerName, Orders.OrderID   
FROM Customers
RIGHT JOIN Orders ON Customers.CustomerID = Orders.CustomerID;
RIGHT JOIN é usados para retornar todas as linhas da tabela à direita.
